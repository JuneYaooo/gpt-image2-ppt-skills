# GPT Image2 PPT -- turn ideas and templates into visual decks

## What I do

GPT Image2 PPT helps users create presentation decks from a topic, outline, or existing `.pptx` template. It drafts the slide copy first, then renders each page as a strong 16:9 visual with an OpenAI-compatible image model, and finally packages the result into an HTML viewer plus a `.pptx` file.

## Best for

- Product launches, pitch decks, strategy reports, courseware, and visual explainers.
- Teams that want a polished first draft quickly, then iterate on copy and style.
- Users who have a reference deck and want a new presentation with similar visual language.

## Key features

- Ten curated visual styles for different presentation moods.
- Template clone mode for adapting an existing `.pptx` design.
- Markdown-first slide planning so copy stays easy to review.
- One-slide smoke tests to control cost and validate visual direction.
- OpenAI-compatible relay support, including providers that expose `gpt-image-2`.

## How it works

1. Tell the expert your topic, audience, page count, and style preference.
2. Review the slide plan in Markdown.
3. Run a single-slide smoke test if needed.
4. Generate the full deck.
5. Open the HTML viewer or use the packaged PowerPoint file.

## Notes

Generated slides are image-based, so they are ready to present but not fully text-editable inside PowerPoint. API keys and provider settings are handled at runtime and are not included in the published expert.
